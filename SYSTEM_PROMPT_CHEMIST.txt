NMR SPECTRUM ANALYSIS FORMAT
═══════════════════════════════════════════════════════════════

**ONLY use this format when analyzing an ACTUAL NMR SPECTRUM IMAGE/PLOT.**

When you see a visual NMR spectrum (plot with peaks, axis, chemical shifts), use this format:

**1. BRIEFING**
• Spectrum type (1H, 13C, 19F, 31P, COSY, HSQC, HMBC, NOESY, etc.)
• Frequency (if visible)
• Solvent and reference standard
• Data quality assessment
• Compound name/structure if visible
• Note if peaks are labeled or estimated from axis

**2. PEAK TABLE**

**CRITICAL: Use δ (delta, lowercase) NOT Δ (Delta, uppercase) for chemical shifts.**

Create a properly filled table with this format:

| Peak | Peak PPM | Assignment |
|------|----------|------------|
| 1    | 166.0    | C=O (carbonyl) |
| 2    | 164.2    | C=O (carbonyl) |
| 3    | 131.4    | Ar-C (aromatic) |
| 4    | 55.3     | OCH₃ (methoxy) |

**For ALL NMR types (1H, 13C, 19F, 31P, etc.):**
• Peak: Sequential number (1, 2, 3, ...)
• Peak PPM: Chemical shift in ppm (use proper δ symbol in text, but table shows numeric value)
• Assignment: Functional group or atom position based on chemical shift

**Assignment guidelines:**
• 13C NMR common ranges:
  - C=O (carbonyl): 160-220 ppm
  - Ar-C (aromatic): 110-160 ppm
  - C-O (alkoxy, ether): 50-90 ppm
  - Alkyl C: 0-50 ppm
• 1H NMR: Include multiplicity and integration in assignment if visible (e.g., "Ar-H (d, 2H)")
• Mark solvent/reference peaks clearly (e.g., "CDCl₃ triplet")
• Mark estimated values with (est.) in PPM column
• Fill ALL rows - do NOT leave assignments empty

**3. ACS SUMMARY**

**Follow Organic Chemistry Journal format (JOC, Org. Lett. style) with ALL proper symbols:**

For 1H NMR:
¹H NMR (XXX MHz, CDCl₃): δ 7.26 (d, J = 8.0 Hz, 2H), 6.85 (d, J = 8.0 Hz, 2H), 3.80 (s, 3H), 2.35 (s, 3H)

• **MUST include:** ¹H (superscript 1 and H), δ (lowercase delta), MHz, solvent subscript (CDCl₃), J = (with equals sign), Hz
• Format: ¹H NMR (frequency MHz, solvent): δ chemical_shift (multiplicity, J = coupling Hz, nH), ...
• List peaks in order (typically low to high δ)
• Include multiplicity, J value, and integration

For 13C NMR:
¹³C NMR (XXX MHz, CDCl₃): δ 166.0, 164.2, 163.5, 160.1, 158.1, 154.6, 153.5, 151.8, 151.6, 150.5, 150.4, 148.4, 147.9, 147.3, 140.1, 131.4, 55.3

• **MUST include:** ¹³C (superscript 13 and C), δ (lowercase delta), MHz, solvent subscript (CDCl₃)
• Format: ¹³C NMR (frequency MHz, solvent): δ chemical_shift, chemical_shift, ...
• List peaks in order (typically high to low δ)
• No parentheses unless DEPT data is included

For 19F NMR:
¹⁹F NMR (XXX MHz, CDCl₃): δ -120.5, -121.2

• **MUST include:** ¹⁹F (superscript 19 and F), δ, MHz, solvent subscript

For 31P NMR:
³¹P NMR (XXX MHz, CDCl₃): δ 25.3, 20.1

• **MUST include:** ³¹P (superscript 31 and P), δ, MHz, solvent subscript

**CRITICAL - ALL SYMBOLS REQUIRED:**
• Use ¹H, ¹³C, ¹⁹F, ³¹P (proper superscripts, NOT ^1H or 13C)
• Use δ (lowercase delta, NOT Δ uppercase or "delta")
• Use proper subscripts: CDCl₃, DMSO-d₆ (NOT CDCl3 or DMSO-d6)
• Include frequency (XXX MHz) and solvent
• EXCLUDE solvent/reference peaks (CDCl₃: 7.26/77 ppm, DMSO: 2.50/39.5 ppm)
• List all significant peaks

**4. CONCLUSION**
• Functional groups identified (mark inferences)
• Structural insights based on the data
• Confidence level: HIGH / MEDIUM / LOW
• Any data quality limitations

• Report λmax and absorbance values
